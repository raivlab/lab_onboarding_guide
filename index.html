미션

기본기를 직접 확인하기 위한 작은 과제입니다.

M1. ROS2

  • 카메라, LiDAR, IMU를 모두 ROS2와 연동
  • 센서 데이터를 topic으로 받아 기록 또는 재생
  • RViz2 또는 rqt로 데이터 흐름과 시각화 결과 정리

M2. Object Detection

  • Ultralytics로 object detection 모델 학습
  • 작은 데이터셋 구성 또는 공개 데이터셋 활용
  • 학습 결과, 예측 이미지, 실패 사례 5개 분석

M3. Feature-Based SLAM

  • Feature-based SLAM 방법 중 하나 실행
  • 입력 데이터셋, 실행 명령, 결과 trajectory 정리
  • 잘 되는 구간과 실패 구간을 각각 분석

M4. LiDAR SLAM

  • LiDAR SLAM 방법 중 하나 실행
  • ROS bag 또는 공개 LiDAR 데이터셋 활용
  • 생성된 map, trajectory, 주요 파라미터 정리

M5. 7-DOF Robot Manipulator

  • Pinocchio와 같은 로봇 라이브러리로 7자유도 로봇팔 모델 로드
  • forward kinematics와 inverse kinematics 계산 코드 작성
  • 로봇팔 자세와 목표 pose를 그래픽으로 시각화
